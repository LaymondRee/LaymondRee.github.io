FPGA Keyboard Piano

  • ⮜

Objective

The goal of this project was to implement a piano using an FPGA, amplifier, and speaker that takes data/clock signals from a PS/2 keyboard as inputs. The requirements for this implementation was to be capable of playing an entire octave from 440 Hz to 880 Hz while only using the 50 MHz clock provided by the FPGA.

Process

The first step in the planning process was to determine all the necessary hardware components and how they would be connected. The PS/2 keyboard clock and data signals would connect to the FPGA. Within the FPGA, digital logic modules will be used to decode the key pressed and output a divided clock signal to match the respective note. This signal will then be amplified before being outputted by the speaker.

All digital logic modules were designed in Quartus Prime using block diagrams or VHDL. The main modules include a clock divider, decoder, and note selector. The clock divider consists of a counter and two comparators that can be configured logic high and low after a specified number of 50 MHz rising edges. The note selection is an 8 input mux that has the frequency of each note as inputs. The decoder uses conditional and case statements in VHDL to determine the key pressed based on the received scancode from the PS/2 data signal.

Each module was simulated in ModelSim before physically implementing the design. For the clock divider, the test divides the clock signal by 10. The note selector and decoder test all eight cases for the input note and scancode. The results for the following simulations are shown below.
