Objective

There were two main goals for this project: design and create a bode plot script for the 2 Series MSO and a configurable first order PCB filter. The requirements for the bode plot script were high accuracy, configurable bandwidth/resolution capabilities, and user friendly for consumer use. The requirements for the filter board were cost effectiveness (under $20) and intuitive configuration settings.

Process

The hardware setup for the bode plot script on other scope models used both the AFG and Auxiliary outputs, however the 2 Series MSO combined both these outputs which meant a different set up was needed. The chosen revised setup would replace the Auxiliary output signal with the AFG output signal. All hardware connections are as follows: AFG connects to CH2 and input of DUT and CH1 connects to the output of the DUT.

The script was written using Python and uses SCPI commands to control the scope. The plan was to compare the amplitude and phase of the AFG output with the DUT output at different frequencies within a selected range. This was done by changing the frequency of the AFG output and adjusting the vertical and horizontal scales for data acquisition. To increase accuracy, synchronization methods and sleep commands were used to ensure stable measurements for each acquisition. Below are script results from low and high pass configurations of varying cutoff frequencies.

A design for the individual low, high, and band pass filters was designed and simulated in LTSpice. Each design only used resistor and capacitor components. The idea was to take advantage of the similarities in designs between filter types by combining them together. By manipulating current paths with switches such that resistors and capacitors are reused would reduce the required number of resistors and capacitors in half.
