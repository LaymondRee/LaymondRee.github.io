DC Power Supply

  • ⮜

Objective

The goal for this project was create a DC to DC power supply that met given customer and engineering requirements regarding cost, safety, accuracy, readability, and configurability. In order to satisfy these requirements, careful consideration when choosing components was necessary throughout the planning process.

Process

The system can be split into six major blocks shown in the top level block diagram below. DC power will be fed into the current control module that will limit current to 1A before feeding the output to the voltage regulator. The Arduino along with the code will interface with the current control, voltage regulator, and LCD to allow user inputted SCPI commmands to control the system output for each individual channel.

The two main circuit elements were for the current control and voltage regulator modules. The current control module uses an LM317 linear regulator to set the maximum output current. The voltage regulator uses two LM2673 buck converters and two MCP41HVX digital potentiometers to set the output voltage of each channel. The Arduino communicates with the digital potentiometers using SPI and outputs the measured voltage to the LCD.

The current control and voltage regulator modules were made into PCBs. Important considerations when designing these PCBs were size, trace width, and ground lead inductance. In order to reduce cost and have a compact design, all PCB elements were placed closely together. Trace width for inputs and outputs were doubled in width to allow 1A to flow without traces overheating. Lastly, all paths to ground were kept short and a ground plane was added to reduce noise caused by ground lead inductance.

Result

The final design mounts all components into a 3d printed enclosure. Bannana plugs are used for input and output of the system. A detailed report of the project can be seen here.
